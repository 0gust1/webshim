Demo/About/Documentation HTML5 shiv

IE8- isn't capable of rendering HTML5 elements. Webshims lib automatically fixes this issue. The only thing you have to do is to add an HTML5 reset stylesheet, wich will add display: block to some of the blockbuilding new HTML5 elements.

Additionally it implements the following Manipulation methods .htmlWebshim(), .appendWebshim(), .prependWebshim(), .afterWebshim() and .beforeWebshim() to also allow dynamic creation of HTML5 elements in IE8-.

$('#my-id').appendWebshim('<section><form action="#"><input type="text" placeholder="hello this is required" required="required" name="test" /><input type="submit" /></form></section>');
