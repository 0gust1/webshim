Introduction

Webshim's forms feautre implements the hole constraint validation API and some other realy nice form features (form-attribute, fieldset[disabled] [list]/datalist, setSelectionRange etc.) in incapable browsers.

The hole specified API can be used to extend and modify those features. This sections only describes some of the build-in extensions to the HTML5 standard.

Locale Settings

Changes to the locale settings normally only affect polyfilled browsers. To get those changes visible in capable browsers either the replaceValidationUI or the iVal (instant/live validation) feature should be used.
