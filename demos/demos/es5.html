Demo/About/Documentation ES5

The es5 feature of Webshims Lib uses the ES5 shim by Kris Kowal.

The script implements the following methods:

  • Object: create, defineProperty, defineProperties, getPrototypeOf, getOwnPropertyDescriptor, getOwnPropertyNames, seal, freeze, preventExtensions, isSealed, isFrozen, isExtensible, keys
  • Array: isArray
  • Array.prototype: forEach, map, filter, every, some, reduce, reduceRight, indexOf, lastIndexOf
  • Date: now, parse
  • Date.prototype: toISOString, toJSON
  • Function.prototype: bind
  • String.prototype: trim

Note: Most of the mentioned Object methods can not be fully implemented. This means, you can only use them with caution. All methods added to the prototype object are visible inside of a for in loop, while native implementations aren't enumerable.

$.webshims.ready('es5', function(){ [1, 2, 3].forEach(function(){ //do stuff }); });
